# 自由合併圖片（Eagle Plugin）

本外掛適用於 Eagle，支援手動拖曳排版與智慧分組，可將多張圖片快速合併為單一輸出圖檔。

![Plugin Logo](./logo.png)

## 功能重點

- 支援 PNG、JPEG、WebP
- 手動排版：可自訂每行格子數量，並拖曳交換位置
- 智慧分組：可依檔名前綴自動分行（可設定每行上限）
- 可先產生預覽，再進行輸出
- 可直接匯入 Eagle 指定資料夾
- 可下載至本機
- 可收藏常用資料夾

## 系統需求

- 已安裝 Eagle（桌面版）
- 外掛目錄需包含 `node_modules`（本外掛依賴 `sharp`）

## 安裝方式

1. 下載 `自由合併圖片.eagleplugin`
2. 於 Eagle 外掛管理頁匯入該檔案
3. 安裝完成後，於 Eagle 啟動外掛

## 使用教學

1. 先於 Eagle 選取欲拼貼之圖片
2. 開啟外掛後，按「重新載入選取」或「追加圖片」
3. 選擇輸出格式（PNG / JPEG / WebP）
4. 選擇排版模式  
手動模式：自行設定每行格子數量，並拖曳圖片至指定位置  
智慧分組：依檔名前綴自動分行，可調整每行上限
5. 按「產生預覽」確認結果
6. 選擇輸出方式  
「匯入 Eagle」或「存到電腦」

## 常見問題

**Q1：匯入或輸出失敗**

- 請先確認已於 Eagle 選取圖片
- 請確認圖片數量與格子容量設定合理

## 隱私說明

- 本外掛不會將圖片上傳至外部伺服器
- 圖片處理於本機完成
- 僅使用本機暫存路徑產生輸出檔案，再匯入 Eagle 或下載
